# AtCoder Favorite Standings

AtCoder の問題ページで、お気に入りユーザーだけに絞った順位表をその場で見られる拡張機能です。

## 機能

- `https://atcoder.jp/contests/*/tasks/*` の問題ページで動作します。
- 問題文の下に、目立ちすぎないタブ風 UI を表示します。
- 問題ページ上でお気に入りユーザー名を追加・削除できます。
- お気に入りはコンテストごとではなくブラウザ全体で共有され、別コンテストでもそのまま使われます。
- 各ユーザーごとに通知の `ON/OFF` 設定を保存できます。
- タブ内の一括管理欄から、複数ユーザーをまとめて編集できます。
- `standings/json` を取得し、お気に入りユーザーだけの順位表を表示します。
- 総合順位、AC 数、時間に加えて、各問題の AC 時刻とペナ数を一覧できます。
- 現在開いている問題の列を薄く強調表示します。
- ページを開いている間は約 30 秒ごとに自動再取得し、`更新` ボタンでも手動再取得できます。
- 通知 `ON` のユーザーは、更新で新しく AC になった問題があればブラウザ通知を出します。

## 注意

- AtCoder にログインしていないと standings の取得に失敗します。
- 旧 AtCoder コンテスト URL では表示だけ対応し、監視取得先は `https://atcoder.jp/contests/<contest-id>/standings/json` を使います。
- 初回読込時は通知せず、前回更新からの差分だけ通知します。

## インストール

### Chrome / Edge

1. `chrome://extensions` または `edge://extensions` を開きます。
2. デベロッパーモードを有効にします。
3. `パッケージ化されていない拡張機能を読み込む` を押します。
4. このリポジトリ内の `extensions/atcoder-user-ac-notifier/` を選択します。

### Firefox

1. `about:debugging#/runtime/this-firefox` を開きます。
2. `一時的なアドオンを読み込む` を押します。
3. このリポジトリ内の `extensions/atcoder-user-ac-notifier/manifest.json` を選択します。

## 使い方

1. AtCoder にログインした状態で対象コンテストの問題ページを開きます。
2. タブを開き、見たいユーザー名を入力して `追加` を押します。
3. お気に入りユーザーだけの順位表がその場に表示されます。お気に入りは同じブラウザ内で全コンテスト共通に保存されます。
